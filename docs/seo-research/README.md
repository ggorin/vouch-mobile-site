## Key Insights Summary

### Market Positioning
- **MVNOs are 42% cheaper** than major carriers with equivalent/better data
- **T-Mobile prepaid** has NO deprioritization (unique competitive advantage AT&T doesn't have)
- **No credit check** market is 70M+ Americans

### Top Target Markets
- **Highest phone bills:** Texas ($205/mo), Pennsylvania ($203/mo)
- **Largest immigrant populations:** CA (26.5%), NJ (23.2%), NY (22.6%)
- **Tech hubs growing:** Austin, Research Triangle, Atlanta, Seattle

### Competitor Weaknesses
- Customer service is major differentiator opportunity
- Hidden fees are #1 pain point
- Data deprioritization poorly explained by competitors

### Content Priorities
1. State pages for high-priority markets (CA, TX, FL, NY)
2. Comparison pages vs top MVNOs (Mint, Visible, Cricket)
3. "No credit check" guide (high search volume)
4. Savings calculator tool (highest engagement potential)

## Source Guidelines

See **[SOURCES.md](SOURCES.md)** for authoritative external links to use in content.

See **[IMAGE-DESIGN-STRATEGY.md](IMAGE-DESIGN-STRATEGY.md)** for visual design guidelines, image specifications, and **comprehensive Gemini/Nanobanana prompting best practices**.

**Use these sources for backlinks:**
- WhistleOut, Tom's Guide, Clark Howard (neutral comparisons)
- FCC, FTC (government/regulatory)
- OpenSignal, Ookla/Speedtest (coverage data)
- Census Bureau, BLS (demographics)
- BBB, ConsumerAffairs (review data - cite stats only)

**DO NOT link to competitor sites:**
- No links to mintmobile.com, visible.com, cricketwireless.com, etc.
- Use neutral comparison sites to discuss competitor features
